📦 Инструкция по запуску:

Распакуйте архив в любую удобную папку на вашем устройстве.
Убедитесь, что установлен Python. Рекомендуется версия 3.12.
При использовании других версий возможны ошибки совместимости.

Откройте терминал (командную строку) и перейдите в папку с игрой.
Например: cd путь/к/папке/с/игрой

Создайте виртуальное окружение:
Windows: py -3.12 -m venv venv
macOS / Linux: python3.12 -m venv venv

Активируйте виртуальное окружение:
Windows: venv\Scripts\activate
macOS / Linux: source venv/bin/activate
(После активации в начале строки терминала должно появиться (venv))

Установите библиотеку для отрисовки: pip install pygame
Дождитесь завершения установки. Если в консоли не появилось ошибок, зависимости установлены корректно.

Запустите игру: python game.py (на Windows также подойдёт py game.py)
